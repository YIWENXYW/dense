std::error_category const& cuda::error_category ( )

Get a reference to the CUDA error category singleton.

std::error_category::name() is overridden to return "cuda". std::error_category::message(int) is overridden to return a CUDA-specific error string for the given int interpreted as a cuda::Error_Code.
